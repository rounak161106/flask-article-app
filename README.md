# 📰 Flask Article Management App

A simple web application built using Flask and SQLAlchemy to manage articles and authors.

## 🚀 Features

- Add new authors
- Add new articles
- Associate articles with authors (many-to-many relationship)
- View all articles
- Filter articles by author

## 🛠️ Tech Stack

- Python (Flask)
- SQLAlchemy (ORM)
- SQLite (Database)
- HTML + Bootstrap (Frontend)
